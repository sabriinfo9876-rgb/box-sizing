Model Box

Content-Box

When you set the width or height of an element, it applies only to the content area. Padding and border are added outside of that width and height, which increases the element's total size.

Total Size Formula:

Total Width = Width + Left Padding +Right Padding +Left Border + Right Border.

Border-box

Is the alternative CSS box model where padding and borders are included inside the element's specified width and height.

Total Size Formula:

Total Width = Specified Width (Padding and Border are inside)
